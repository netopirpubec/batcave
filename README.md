# batcave

Custom Claude Code skills, installable via [skills.sh](https://skills.sh).

## Install

```sh
npx skills@latest add netopirpubec/batcave
```

## Skills

- **domain-triage** — triage a queue (issues, tickets, leads, incidents, PRs) through a state machine driven by a domain-knowledge brief passed as the skill argument.
- **domain-grill** — Socratic deep-dive grilling against a markdown brief you pass as the first argument. A senior-practitioner persona uses the brief as ground truth and drills one topic at a time to expose gaps.
